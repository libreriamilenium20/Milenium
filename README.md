📚 Librería Milenium - Santiago Tianguistenco
¡Bienvenidos al repositorio oficial de la plataforma web de Librería Milenium! Este sitio web funciona como un catálogo interactivo digital y Sistema de Gestión de Contenidos (CMS) diseñado específicamente para nuestra sucursal física.

🚀 Características del Proyecto
Catálogo Dinámico: Filtrado de libros en tiempo real por texto o categorías.
Buscador Inteligente: Sistema de autocompletado automático (estilo Google) para facilitar la búsqueda al cliente.
Panel de Administrador Integral: Interfaz protegida para añadir, editar, dar de baja libros o cambiar categorías sin tocar el código.
Diseño Personalizable: Selector de colores de identidad y carga de logotipo/QR directo desde el panel de control.
Fondo Animado: Efecto interactivo de partículas de libros flotantes mediante HTML5 Canvas.
🛠️ Tecnologías Utilizadas
HTML5 (Estructura de la página y maquetado)
CSS3 (Estilos premium, rejilla CSS Grid, flexbox y variables dinámicas)
JavaScript (Vanilla) (Motores de búsqueda, ordenamiento de inventario y renderizado dinámico)
FontAwesome (Iconografía de redes sociales y controles del panel)
